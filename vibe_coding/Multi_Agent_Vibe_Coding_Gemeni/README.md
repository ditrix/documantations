# Мультиагентный вайб-кодинг (Multi-Agent Vibe Coding)

Документация по методологии мультиагентного вайб-кодинга — эволюции классического программирования, где акцент смещается с написания строк кода на управление потоками мыслей и взаимодействием нейросетевых агентов.

## Содержание документации

### Основные разделы

- [Введение в мультиагентный вайб-кодинг: принципы, инструменты и примеры](Part_1.md)  
  Основные принципы, инструменты (Cursor, Cline, CrewAI), пример разработки блога с комментариями, подготовка и развертывание проекта.

- [Мультиагентный подход: роли вместо разных чатов](Part_2.md)  
  Объяснение, почему мультиагентный подход — это не открытие разных чатов, а использование ролей через инструкции в Cursor. Работа через Rules и режим Composer.

- [Практическое руководство: настройка Cursor для мультиагентной разработки](Part_3.md)  
  Конкретный план настройки для разработки блога: создание центра управления (.cursor/rules/), файлы инструкций для агентов (architect.mdc, coder.mdc, ui-specialist.mdc), алгоритм действий.

- [Готовый мастер-промпт для развертывания проекта](Part_5.md)  
  Готовый промпт для режима Composer (Ctrl+I) для инициализации проекта "Vibe-Blog" на стеке Next.js, Prisma и Tailwind CSS с пошаговыми инструкциями для каждой роли.

- [Вайб-кодинг для React + TypeScript: Type-First Development](Part_7.md)  
  Адаптация методологии для стека React + TypeScript: Type-First Development, добавление авторизации и редактирования, технические нюансы Fullstack-разработки.

- [Детальная реализация на React + TypeScript + Next.js](Part_8.md)  
  Подробное руководство по реализации вайб-кодинга на React + TypeScript + Next.js: подготовка среды, запуск задач, визуализация процесса, подготовка и развертывание.

### Итоговые руководства

- [Краткое руководство по мультиагентному вайб-кодингу (TS/React)](Total_1.md)  
  Структурированное руководство по методологии: подготовка среды, создание MVP блога, расширение функционала, развертывание и контроль качества.

- [Финальный документ-шаблон: манифест вайб-кодинга](Total_2.md)  
  Готовый шаблон-манифест для старта любого нового React-проекта в Cursor: конфигурация правил, этапы разработки, контроль качества и развертывание.

## Структура документации

Документация организована в логической последовательности:

1. **Теоретическая база** (Part_1, Part_2) — введение в концепцию и принципы работы
2. **Практические инструкции** (Part_3, Part_5) — настройка и использование инструментов
3. **Специализированные руководства** (Part_7, Part_8) — адаптация под конкретные стеки
4. **Итоговые материалы** (Total_1, Total_2) — краткие справочники и шаблоны
